# Security Policy

## Supported Versions

Only the latest stable version of Laravel Livewire Tables is supported for security patches.

| Version | Supported          |
| ------- | ------------------ |
| 3.x     | :white_check_mark: |
| 2.x     | :x:                |
| 1.x     | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this package, please send an e-mail to **skywalkerlknw@gmail.com**. All security vulnerabilities will be promptly addressed.

Please provide:
- A description of the vulnerability.
- Steps to reproduce (including a proof-of-concept if possible).
- Any potential impact details.

**Please do not use the GitHub issue tracker for security-related reports.**

We will acknowledge your report within 48 hours and provide an estimated timeline for a fix.
